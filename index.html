    • Grand Market

    Bukkit plugin to create a server-wide marketplace with flexible prices.

    Project maintained by GrandMarket Hosted on GitHub Pages — Theme by mattgraham

    The Grand Market is a plugin for Bukkit Minecraft Servers, that manages a server-wide marketplace.
    Users are able to buy and sell items from each other, at prices that they agree on.

    Currently the project is under development, and is not ready for use yet.

    Features

    • Global Marketplace
      All users can trade items, from any location
    • Flexible Prices
      Users can mark how much they want to buy/sell an item for
